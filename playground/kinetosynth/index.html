Cómo Usar

Pinza (pulgar + índice) para agarrar y mover objetos
Rota mientras sostienes para cambiar parámetros
Arrastra Generadores al centro para activar el sonido
Coloca Efectos entre generadores y el centro para procesar audio
LFO modula objetos cercanos compatibles
Presiona "R" para resetear todos los objetos
Generadores (Pad, FM, Bells, Pulse)
Efectos (Delay, Reverb)
Controladores (LFO verde, LFO2 rojo)
Global (Tempo - rota para cambiar BPM)
